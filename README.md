# commits

Este repositório é gerenciado pelo [Commitly](https://commitly.audibert.dev), uma ferramenta que registra seus commits de forma automática e organizada.

## Como funciona

Cada commit registrado pelo Commitly é salvo no arquivo `changes.json` deste repositório, mantendo um histórico completo das suas atividades.

> [!WARNING]
> Se este repositório for deletado, todos os commits feitos nele deixarão de aparecer na sua atividade.

## Contribuições

O projeto é open source e contribuições são muito bem-vindas!

Se você quiser sugerir melhorias, reportar bugs ou contribuir com código, você pode:

- Abrir uma issue relatando problemas ou sugerindo features;
- Propor melhorias na experiência ou novas ideias;
- Enviar um Pull Request com sua contribuição.

Toda ajuda é bem-vinda para tornar a solução cada vez melhor.

> _Considere deixar uma estrela ⭐ no [projeto open source do Commitly](https://github.com/matheusaudibert/commitly) se achar a ferramenta útil!_
